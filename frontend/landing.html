How It Works

三步完成简历筛选

简单直观的操作流程，无需复杂培训，HR 上手即用

1

填写岗位 JD

粘贴或输入岗位描述，设置自定义匹配维度（工作年限、学历要求、技能权重等）。

2

上传简历文件

拖拽或点击上传简历，支持批量上传最多 10 份，自动完成解析、OCR 识别与信息脱敏。

3

查看筛选结果

按匹配度从高到低排序展示，查看每份简历的详细分析，支持导出 Excel，高效推进面试。

Privacy & Security

数据安全，本地优先

我们深知简历数据的敏感性，从架构设计上保障你的数据安全

🏠

本地部署运行

工具完全在本地运行，简历数据仅存储在你的电脑上，不会上传至任何第三方服务器，从源头保障数据安全。

🔐

API Key 加密存储

使用 Fernet 对称加密算法存储 API Key，密钥独立保存在本地配置目录，不明文存储、不明文传输。

🛡️

脱敏后再分析

简历内容在发送给大模型前，自动完成敏感信息脱敏处理，只传输必要的工作经历、技能等非隐私信息。
